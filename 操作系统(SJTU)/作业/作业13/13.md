##### 13.7 Consider a system that supports 5,000 users. Suppose that you want to allow 4,990 of these users to be able to access one file.

- How would you specify this protection scheme in UNIX?
  - 为这4990个用户创建一个用户组，这个用户组有权限访问这个文件
  - 创建一个访问控制列表，里面包含这4990个用户
- Can you suggest another protection scheme that can be used more effectively for this purpose than the scheme provided by UNIX?
  - 因为只有10个用户不能访问，只需要将这10个用户加入访问控制列表，这个访问控制列表里的用户不能访问这个文件
